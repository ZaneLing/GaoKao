# AI安徽高考志愿填报系统

安徽高考数据智能分析平台，基于多Agent架构实现智能志愿推荐。
![banner](p.png)

## 功能特性

- **了解学校** - 分析指定学校的综合信息（历年分数线、招生计划、专业详情等）
- **志愿推荐** - 根据考生排名和专业意向，制定冲稳保志愿方案，生成最终填报表格
- **多Agent协作** - 分数线分析、院校信息、专业组分析、策略推荐四大Agent协同工作
- **数据可视化** - 表格形式展示志愿方案，支持15-45个志愿推荐

## 志愿推荐逻辑与流程

### 志愿推荐核心思路

本系统采用 **"排名法"** 而非"分数法"作为核心推荐依据：

```
排名法优势：
  - 每年高考难度不同，但排名相对稳定
  - 用位次对标往年数据更准确
  - 避免因题目难度导致的分数波动
```

### 推荐流程（多Agent协作）

```
┌─────────────────────────────────────────────────────────────────┐
│                    志愿推荐完整流程                              │
└─────────────────────────────────────────────────────────────────┘

第1步：用户输入
──────────────────────────────────────────────────────────────────
  输入：全省排名 + 意向专业 + 意向城市（可选）+ 院校偏好（可选）
  
第2步：分数线分析（ScoreAgent）
──────────────────────────────────────────────────────────────────
  • 根据排名换算等效分数
  • 对标2024年各院校专业录取位次
  • 筛选出"有希望"的院校范围
  
第3步：院校信息分析（SchoolAgent）
──────────────────────────────────────────────────────────────────
  • 查询院校基础信息（985/211/城市）
  • 过滤意向城市、院校偏好
  • 评估院校综合实力
  
第4步：专业组分析（MajorGroupAgent）
──────────────────────────────────────────────────────────────────
  • 按专业相关性分组
  • 确保所有推荐专业与意向专业相关
  • 计算专业组内录取概率
  
第5步：策略推荐（StrategyAgent）
──────────────────────────────────────────────────────────────────
  • 制定冲稳保策略
  • 冲：排名略高于往年录取位次（概率40%-60%）
  • 稳：排名与往年录取位次相当（概率60%-80%）
  • 保：排名远高于往年录取位次（概率80%+）
  
第6步：生成最终方案
──────────────────────────────────────────────────────────────────
  • 按比例选取：冲5 + 稳15 + 保25 ≈ 45个志愿
  • 生成志愿填报表格
  • 输出详细分析报告
```

### 冲稳保策略说明

| 策略 | 数量建议 | 录取概率 | 说明 |
|------|---------|---------|------|
| 冲一冲 | 5个 | 40%-60% | 尝试冲击比自身排名更高的院校 |
| 稳一稳 | 15个 | 60%-80% | 录取概率较大的目标院校 |
| 保一保 | 25个 | 80%+ | 确保有学可上的稳妥院校 |

### 关键约束条件

1. **专业相关性** - 所有推荐专业必须与用户意向专业相关
   - 例如：用户选择"数学"，则推荐专业组都包含数学相关

2. **城市偏好** - 可选筛选条件
   - 如选择"合肥"，则优先推荐合肥院校

3. **院校层次** - 可选筛选条件
   - 如选择"211"，则过滤非211院校

4. **志愿数量** - 最多45个，至少15个推荐

## 技术特点

1. **模块化设计** - 各功能独立为Skill，可单独调用和测试
2. **多Agent协作** - 不同Agent负责不同分析任务，结果汇总输出
3. **智能志愿推荐** - 基于冲稳保策略，结合专业相关性约束
4. **数据自动整合** - 从多个Excel文件提取数据，生成统一报告
5. **输出保存** - 完整日志和分门别类的数据文件便于事后回顾

## 注意事项

- 排名是主要对标往年数据的指标，应优先填写
- 意向专业是必填项，所有推荐将围绕该专业展开
- 志愿数目建议15-20个，最多45个
- 数据来源于安徽省教育考试院，仅供参考
