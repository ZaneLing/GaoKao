# AI辅助高考志愿填报系统

安徽高考数据智能分析平台，基于多Agent架构实现智能志愿推荐。

## 功能特性

- **了解学校** - 分析指定学校的综合信息（历年分数线、招生计划、专业详情等）
- **志愿推荐** - 根据考生排名和专业意向，制定冲稳保志愿方案，生成最终填报表格
- **多Agent协作** - 分数线分析、院校信息、专业组分析、策略推荐四大Agent协同工作
- **数据可视化** - 表格形式展示志愿方案，支持15-45个志愿推荐

## 脚本说明

### 核心脚本

| 脚本 | 说明 |
|------|------|
| `main.py` | **主入口程序**，启动系统主菜单，支持功能选择（了解学校/志愿推荐） |
| `agent.py` | Agent框架核心，实现GaokaoAgent（单Agent）和MultiAgentSystem（多Agent） |
| `skills.py` | 技能调度模块，统一管理和调用各技能 |

### Skills技能模块

位于 `skills/` 目录：

| 模块 | 说明 |
|------|------|
| `score_line_skill.py` | 分数线分析技能：历年分数线、一分一段、批次线等 |
| `school_info_skill.py` | 院校信息技能：基础信息、院校对比、录取分析 |
| `enrollment_plan_skill.py` | 招生计划技能：招生章程、计划数、专业组计划 |
| `major_info_skill.py` | 专业信息技能：专业详情、专业对比、专业就业分析 |
| `major_group_skill.py` | 专业组分析技能：专业组划分、选科要求、志愿推荐 |
| `school_analysis_skill.py` | 学校综合分析技能：整合多源数据生成完整报告 |
| `reference_data_skill.py` | 参考数据技能：学科评估、大学排名、保研率等 |

### 数据检查脚本

| 脚本 | 说明 |
|------|------|
| `check_data_structure.py` | 检查Excel数据文件结构（保留备用） |

## 配置说明

### 环境要求

- Python 3.8+
- openpyxl（Excel处理）
- 配置好API密钥（见 `.env` 文件）

### 数据更新

如需更新数据，将新的Excel文件放入对应的数据目录：

- 分数线数据 → `安徽/1_最近年份数据/分数线/`
- 招生计划 → `安徽/1_最近年份数据/招生计划/`
- 院校信息 → `安徽/3_院校信息/`
- 专业信息 → `安徽/4_专业信息/`

## 技术特点

1. **模块化设计** - 各功能独立为Skill，可单独调用和测试
2. **多Agent协作** - 不同Agent负责不同分析任务，结果汇总输出
3. **智能志愿推荐** - 基于冲稳保策略，结合专业相关性约束
4. **数据自动整合** - 从多个Excel文件提取数据，生成统一报告
5. **输出保存** - 完整日志和分门别类的数据文件便于事后回顾

## 注意事项

- 排名是主要对标往年数据的指标，应优先填写
- 意向专业是必填项，所有推荐将围绕该专业展开
- 志愿数目建议15-20个，最多45个
- 数据来源于安徽省教育考试院，仅供参考
