# 安徽高考资料整理说明

## 文件夹结构

本文件夹已重新整理为以下5个主要分类：

### 1️⃣ **1_最近年份数据/** - 最新的报志愿参考数据
最适合用于当前年度的高考志愿填报参考。

- **招生计划/** - 2021-2025年度招生计划数据
  - 包含：招生计划人数、专业计划等
  
- **分数线/** - 2024-2025年最新分数线数据
  - 包含：一分一段、批次线、院校分数、专业分数等
  
- **选科信息/** - 2024年选科要求
  - 包含：各院校各专业的选科限制

### 2️⃣ **2_历史数据/** - 历史参考数据
用于分析历年趋势和参考。

- **2020年及以前/** - 2008-2020年历史数据
  - 包含：各年份院校分数、提前批分数等
  
- **2021年数据/** - 2021年度完整数据
  - 包含：招生计划、分数线、招生章程等
  
- **2022年数据/** - 2022年度完整数据
  - 包含：招生计划、分数线、招生章程、排名等

### 3️⃣ **3_院校信息/** - 院校基本信息
- 院校基础信息（联系方式、地址等）
- 院校专业分数线汇总数据库

### 4️⃣ **4_专业信息/** - 专业详细信息
用于了解专业发展、就业前景等。

- 专业介绍及薪酬数据
- 专业就业分析（地区、行业、职业方向）
- 专业排名信息
- 院校及专业满意度
- 重点专业、特色专业

### 5️⃣ **5_参考数据/** - 行业参考数据
用于深度了解高校整体情况。

- 大学排名（QS、U.S.News、校友会、泰晤士等）
- 学科评估（第四轮全国高校学科评估）
- 保研资格院校统计
- 本科专业目录
- 高考院校电话等

---

## 使用建议

### 📋 填报志愿前
1. 从 **1_最近年份数据** 查看最新的招生计划和分数线
2. 从 **4_专业信息** 了解感兴趣的专业信息
3. 从 **3_院校信息** 查看院校基本信息

### 📊 对比分析
1. 使用 **2_历史数据** 中的历年数据进行趋势分析
2. 查看分数线、招生人数的变化情况

### 🎓 深度研究
1. 参考 **5_参考数据** 中的排名和学科评估
2. 通过就业数据了解专业前景
